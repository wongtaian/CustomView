# 各种自定义 View. 
## 一、图表. 
### 1、3D立体饼图. 
  3D立体的饼图，可自定义饼图的开始角度。  
  从内到外的原则，依次画底部椭圆，背部立面，内部矩形，外部立面，顶部椭圆，顶部滑过的扇形.
